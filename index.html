SpringBootCareRecord

介護記録用アプリ（SpringBoot使用）。
使用言語 - Java
1つ上のアプリをSpringBootを使用してブラウザ上で扱えるようにしたアプリです。
グループホーム等の介護記録を簡単に行える様にと作成しています。
記録したデータをExcelファイルとして出力する事も出来ます。

介護記録用アプリSP.zip ダウンロード

ダウンロード後、zipファイルを解凍し、フォルダ内の 使用方法.txt ファイルに書かれている方法でアプリを開始してください。
